# pokemongodex
Checklists to help keep track of your Pokemon Go Pokedex in a way that friends can also see.

This was made with help from a friend of mine, though much of the actual work was done by me. 
Info on various aspects of these sheets including which Pokemon are available in the game, whether they can be shiny, traded, shadow, etc. was found with searching Google.
All links used for the costume info is included in that particular sheet.
The idea and format for the checkboxes (originally made on Google Sheets) came from another spreadsheet I found made by jaynathias.

My friend and I wanted a different way to keep track of the Pokemon we had in Pokemon Go, along with a way to see what others had so we could save Pokemon for trades as well as compare completetion, and thus these spreadsheets were born.
The Pokemon Go Dex sheet (the one with xxl, xxs, shadow, etc.) was made much more recently than the others.
All sheets are pre-set-up for a certain amount of people to use (2 in Pokemon Go Dex, 6 in Shiny Living Dex, and 5 in both Costume and Lucky Living Dex), if you want to use with more people, you'll just need to select and drag over the checkboxes, or copy and paste.

The Pokemon Go Dex spreadsheet includes 6 sections not originally included in the other sheets and is formatted a little differently.
The Mega/Gigantamax section includes every Mega, Gigantamax, Primal, and other special forms I felt like including such as the fusions and enhanced forms.
  This section also includes all forms that exist in the main series, but haven't been released into Pokemon Go.
  There are 2 columns for each person, one for regular, and one for shiny.
  All of the forms that haven't been released into the game yet have been grayed out, other than Zygarde's shiny form, which I just realized that I hadn't grayed out.
  Correct as of the release of Mega Raichu.
The 100%, XXL, and XXS sections are exactly as they say and include all Pokemon that are in the main series games.
  These sections do not include any regional forms or any of the multiple form Pokemon (such as Unown or Vivillon), this means that only one entry for each Pokemon exists in the lists to simplify them.
  Each main series generation is denoted by an arbitrarily chosen color that is only there to denote a divide between generations so you can find things easier, although searching the sheet for the Pokemon you need is a lot easier anywways.
  Pokemon that haven't been released into Pokemon Go have been grayed out as usual.
  Correct as of the release of Zeraora.
The Purified and Shadow sections are exactly as they say and include all Pokemon that are in the main series games.
  This one is similar to the one above, though much more is grayed out as many Pokemon have not been released as Shadow in Pokemon Go.
  The colors are the same as the previous sheet.
  Correct as of July 19, 2026.

The Shiny Living Dex Sheet includes all Pokemon that are in the main series games, and has been split up into smaller sheets by main series generation.
  This includes all alternate forms of the Pokemon other than Megas, Gigantamax, Primals, Fusions, and other Enhanced forms (such as Zygarde, Zacian, and Zamazenta, which are included in the first sheet).
  All Pokemon not released into Pokemon Go have been grayed out.
  This was designed to be used as a living dex checklist, but of course can be used without requiring a living dex.
  Correct as of July 19, 2026.

    (The rest of the sheets to be added later)

Lastly, I will not be updating these sheets unless there is a major change that needs to be dealt with, such as a new main series game release, so when various Pokemon get their first release in Pokemon Go, it will be up to you to add stuff if you wish to use this sheet.
